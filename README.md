# ExpressJS + Socket.IO + AngularJS App

## Getting started
  1. npm install
  2. jspm install
  3. node app.js
  4. open with your browser http://localhost:3000
  4. optional: gulp (compiles scss with a file watcher)"


based on tuts from Frederik Dietz
